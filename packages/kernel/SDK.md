## Ableton Extensions SDK

This package bundles a Live extension. Install Ableton's Extensions SDK locally (not redistributed here):

1. Download `@ableton-extensions/sdk` and `@ableton-extensions/cli` from Ableton's Extensions SDK distribution.
2. `npm install /path/to/ableton-extensions-sdk-*.tgz`
3. `npm install -D /path/to/ableton-extensions-cli-*.tgz`

CI on forks skips kernel packaging when SDK tgz files are absent.
